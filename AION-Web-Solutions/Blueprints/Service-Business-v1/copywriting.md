# Copywriting Rules — Service Business v1

## Hero Formula
Always: **Problem + Solution + Trust Element**

## H1 Format
```
[Service] [Category] & [Category] με [Trust Word]
```

### Examples
- "Επισκευή Laptop, Desktop & Mac με Εγγύηση"
- "Εγκατάσταση & Συντήρηση Κλιματιστικών με Εγγύηση"
- "Security & Δορυφορικά Συστήματα με Πιστοποίηση"

### Rules
- Last word is styled in accent color (green)
- Never start with "Welcome to" or company name
- No generic titles ("Our Services", "Quality Work")
- One clear promise, not multiple messages
- Max 10-12 words

## Subtitle
- Address pain point + mention solution
- 15-25 words max
- Tone: helpful, not salesy
- Example: "Άμεση διάγνωση, γνήσια ανταλλακτικά και εξειδικευμένη τεχνική υποστήριξη."

## CTA Buttons
| Type | Label | Action |
|---|---|---|
| Primary | "Καλέστε Τώρα" / "Ζητήστε Προσφορά" | `tel:` link |
| Secondary | "Δωρεάν Διάγνωση" / "Δωρεάν Εκτίμηση" | scroll to contact |

Never use generic labels: "Learn More", "Click Here", "Submit".

## Trust Metrics
Always use numbers + short labels:
- 100% → Γνήσια Ανταλλακτικά
- 24-48h → Χρόνος Επισκευής
- 6 Μήνες → Εγγύηση
- Adaptation: 15+ Χρόνια, 500+ Πελάτες, 1.000+ Επισκευές

Font: Big number (bold) + smaller label (uppercase, tracked)

## Services
- Write from customer pain, not service name
- "Το laptop σας δεν ανοίγει;" → "Επισκευή Laptop"
- Keep descriptions ≤25 words, benefit-focused
- Use local photos over stock (client workspace if possible)

## Process
- 4 steps maximum
- Action verbs: Παραλαβή → Διάγνωση → Επισκευή → Παράδοση
- Numbered badges for scannability
- One line connecting them visually (desktop only)

## FAQ
- Write questions as the CUSTOMER would search/ask
- Minimum 9 questions (for rich snippet eligibility)
- One question must address service area/range
- Answers: 1-3 sentences, direct, no fluff
- Include schema FAQPage JSON-LD

## Service Area
- One section, not multiple landing pages
- Natural copy, no keyword stuffing
- "Εξυπηρετούμε πελάτες από [location] και όλη την ευρύτερη περιοχή της Αθήνας, όπως [area list]"
- Mention 5-8 surrounding areas
- Also include in Footer + FAQ as natural anchors

## Social Proof
- Star rating + number of reviews + short headline
- "4.9/5 — Βασισμένο σε 50+ αξιολογήσεις"
- External link to Google Reviews
- Displayed high on page (before Services)

## Case Studies (Optional)
Only include if client has real photos + data:
- Problem → Diagnosis → Solution → Result
- Include measurable outcomes (temps 95°C → 68°C, repair in 24h)
- Never fabricate — credibility is fragile
